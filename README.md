# Bash aliases
Some useful bash aliases:
 - General utility stuff
 - Docker
 - NPM
 - Laravel
 - Git
 - Composer

# Installation
Load your bash aliases in the .bashrc file

```
if [ -f ~/.bash_aliases ]; then
. ~/.bash_aliases
fi
```
